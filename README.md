# Polymarket · Copa do Mundo

Tracker de apostas da Copa do Mundo 2026 via Polymarket, com controle de bankroll, juros compostos e conversão USD/BRL em tempo real.

## Funcionalidades

- Registro de apostas com partida, odd, valor e categoria
- Cotação USD/BRL em tempo real (AwesomeAPI)
- Dashboard com KPIs: bankroll, lucro/prejuízo, ROI, taxa de acerto, sequência
- Projeção de crescimento por juros compostos
- Gráfico de evolução do bankroll
- Exportação para CSV
- Persistência local (localStorage)
- Design Apple minimalista

## Como usar

Abra o arquivo `index.html` em qualquer navegador — sem necessidade de servidor.

Ou acesse via GitHub Pages em:  
`https://leofrancasilva12.github.io/polymarket-copa-do-mundo`

## Stack

- HTML5 / CSS3 / JavaScript vanilla
- Canvas API para gráficos
- AwesomeAPI para cotação de câmbio
- localStorage para persistência
